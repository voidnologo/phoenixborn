# Game Night

**Seven constructed decks, all built from our one shared collection.** This is the set that inspired the whole project — Jayson Seth Lindley's *"Seven Constructed Ashes Decks From One Collection"* — and now that the last three expansions (The Scholar of Ruin, The Ocean's Guard, The Messenger of Peace) are in the binder, we can finally build every one of them. Each leans on its own Phoenixborn and its own magic, so they barely reach for the same cards.

Three people playing tonight? Grab any three that don't share a token pile and you're set — there's a **recommended clean trio** at the bottom, plus the full **build-together matrix** for rotating more decks in over the night. Every deck below has a **Starting Five**, a one-line **win condition**, a **game plan**, and an expandable **Build list** showing exactly which prebuilt box each card comes from — the fast way to pull cards from a binder sorted by Phoenixborn. Each also links to its full list on [ashes.live](https://ashes.live), where you can clone it into [ashteki](https://ashteki.com) to play online.

!!! info "Reading the build lists"
    Cards are grouped by the **prebuilt box you pull them from**, so you visit each box once. A card's box is the Phoenixborn whose deck it shipped in (the *Master Set* holds six starters, filed under each starter Phoenixborn). Conjurations aren't listed separately — they come in the same box as their **Summon** card.

## Building several at once — the matrix

Every 30-card list divides cleanly across the collection: the shared **standard** staples all fit at once (several land at exactly three-of-three — Nature's Wrath, Final Cry, Fire Archer, Sympathy Pain, Fate Reflection, Era's End, Excavate). The only limit on building all seven simultaneously is a handful of shared **conjuration token piles** — the physical tokens two decks would both want. Any two decks marked ✅ build together with zero bookkeeping; ⚠️ means they draw from the same token pile, so pick one to field it (or split the tokens).

Tristan (the Archer) is the one deck that collides with several others — it wants the Prism Tetra, Frostback Bear/Gilder, and Ruby Cobra piles that four other decks also use. Everything else is nearly free of conflict.

!!! tip "Tonight's three-player table"
    For the first game night, grab **Maeoni (Enchantress) · Aradel (Beastmaster) · James (Assassin)** — all mutually ✅, and a clean spread of aggro, toolbox-grind, and tempo. Add **Victoria (Illusionist)** for a fourth that still conflicts with none of them. That's the largest set that builds all at once with no token sharing; to rotate any of the other three in, just consult the ⚠️ cells above and hand off the one shared pile.
