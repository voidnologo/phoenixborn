# Completing the Collection

*Where the binder stands, built by cross-referencing the 30 owned Phoenixborn against the complete Ashes release history (all 31 releases in the ashes.live database, spanning the original 1.0 line, Ashes Reborn, and Ashes Ascendancy).*

## The short version

The **PvP constructed collection is now complete**: all **28 player-versus-player Phoenixborn** from the original 1.0 / Ashes Reborn line, plus the Ascendancy Starter. With the last three deluxe expansions added, there is nothing left to buy for a fully complete constructed pool. What remains is entirely optional or not-yet-released:

1. ~~**Three PvP constructed expansions**~~ — **done.** The Scholar of Ruin, The Ocean's Guard, and The Messenger of Peace are now in the binder.
2. **The Ascendancy expansion cycle** — new content built on the two new magic types, only just beginning to release.
3. **The entire Red Rains co-op line** — optional, since the group plays PvP, but the only way to get solo/cooperative Ashes.

!!! info "How the eras fit together"
    The original **1.0** line (2016–2019) and **Ashes Reborn** (2021+) contain the *same* Phoenixborn — Reborn re-released the 1.0 catalog with revised cards (the job the 1.5/2.0 upgrade kit did). There's no reason to chase the old 1.0 white boxes for play; the Reborn/Ascendancy versions are tournament-legal. **Ashes Ascendancy** (2024+) is the current starting point; when it launched, the Reborn Master Set and all expansions moved to **print-on-demand** (still legal, just off the shelf).

The last three standard expansion decks are now owned. Each added a Phoenixborn and its full card set, and — importantly — they unlock the well-known **"seven decks from one collection"** builds that previously needed cards the binder lacked. That set is now live as the [Game Night](game-night.md) lineup.

## 2. Ashes Ascendancy expansions — new content, just starting

The **Ascendancy Starter Set** (Arren Frostpeak + Issa Brightmore, introducing **Astral** and **Artifice** magic) is owned. Ascendancy is a planned seven-set cycle; each set pairs the two new magic types with an older dice color and adds two Phoenixborn plus a co-op Dragonborn boss. As of mid-2026 only the first has released:

| Expansion | Phoenixborn | Status |
|---|---|---|
| **Ascendancy: Natural Magic Expansion** | Maeve Luminvale + Kanna | Released ~Q2 2026 — **currently available** |
| Remaining Ascendancy cycle (~6 more sets) | 2 new Phoenixborn each | Not yet released — buy as they ship |

## 3. Red Rains — solo & co-op (optional)

Red Rains is the cooperative/solo mode: one or two players fight an AI-run **Chimera** boss. It's a different experience from PvP game nights, so treat it as nice-to-have. Each set also bundles a mono-magic preconstructed deck and new "second unique" cards for four existing Phoenixborn. There are seven sets:

| Red Rains Set | Magic | Notes |
|---|---|---|
| The Corpse of Viros | Ceremonial | The entry point; required to play *The End of Argaia*. |
| The Frostwild Scourge | Natural | Bundled the first errata wave. |
| The Blight of Neverset | Charm | |
| The Spawn of Shadowreck | Illusion | Bundled the second errata wave. |
| The Siege of Lordswall | Divine | |
| The Flood of Moon Cove | Sympathy | |
| **The End of Argaia** | Ruin/Rebirth Chimera | The finale of the Red Rains cycle and the last "Reborn"-branded release; play Jericho, Rowan, Tristan, or Orrick against a new boss. Requires the Master Set + The Corpse of Viros + 10 Time dice. |

*Availability: print-on-demand or secondary market. To try co-op, start with **The Corpse of Viros**.*
